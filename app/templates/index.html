PRNG

Pseudo Random Number Generator API.


URL         : /prng                                             
METHOD      :   GET                                             
URL PARAMS  :                                                   
[type]  - uint8/uint16/hex16                                    
[length]- 1-1024                                                
[size]  - 1-1024 [only if hex16 is selected]                    
                                                                  
example:                                                        
/prng?type=uint16&length=10                                      
{"data": [64723], "length": 1, "type": "uint16", "success": true}
